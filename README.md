## Tech Stack

**Client:** Next.js, styled-jsx

## Setup

```bash
$ cd movie-app-nextjs/
$ cp .env.sample .env

# Add your TMDB API Key in .env file

$ npm install
$ npm run dev

# http://localhost:3030/
```
