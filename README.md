## Vercel deployment URL
https://movie-app-nextjs-orpin.vercel.app/

## Tech Stack

**Client:** Next.js, styled-jsx

## Setup

```bash
$ cd movie-app-nextjs/
$ cp .env.sample .env

# ADD your Add your TMDB API Key in .env file

$ npm install
$ npm run dev

# http://localhost:3030/
```
